# 🔐 PicoCTF Writeups & Practice

Welcome to my **PicoCTF repository**, where I document my journey through Capture The Flag (CTF) challenges, focusing on practical cybersecurity skills, problem-solving, and real-world attack techniques.

---

## 📌 About PicoCTF

[PicoCTF](https://picoctf.org/) is a beginner-friendly cybersecurity competition designed to teach fundamental concepts in:

* Cryptography
* Web Exploitation
* Reverse Engineering
* Binary Exploitation
* Forensics
* General Skills

---

## 🎯 Purpose of This Repository

This repository serves as:

* 📘 A structured collection of solved challenges
* 🧠 A personal knowledge base for revision
* 🚀 A showcase of my cybersecurity learning journey
* 🔍 A reference for beginners starting with CTFs
